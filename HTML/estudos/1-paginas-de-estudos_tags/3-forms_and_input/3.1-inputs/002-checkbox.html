"CHECKBOX"

Definição

A input <checkbox> Cria uma caixa de seleção que é renderizada por padrão como caixas que são "marcadas" quando ativadas.

A aparência exata depende da configuração do sistema operacional sob qual o navegador está sendo executado.

Geralmente é um quadrado, mas pode ter cantos arredondados. Uma caixa de seleção permite selecionar valores únicos para envio em um formulário (ou não)

Maneiras de inserir a input "checkbox":

                            <input type="checkbox" name="checkbox" value="checkbox id"button" checked>

Atributos Globais

A input <button> suporta todos os Atributos Globais HTML.

accesskey, class, contenteditable, data-*, dir, draggable, hidden, id, lang, spellcheck, style, tabindex, title, translate.

Atributos Input

type, name, value, checked.
