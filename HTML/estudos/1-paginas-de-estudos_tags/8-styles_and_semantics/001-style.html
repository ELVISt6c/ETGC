<STYLE>

Definição

A tag <style> </style> define o estilo (CSS) em um documento (HTML).

Dentro do elemento deve ser especificado os termos HTML que serão renderizados em um navegador.

A tag abrange folhas de estilo para uma página web.

Importante: Há dois tipos de tag <style> </style> em um documento HTML.

Importante: O segundo tipo de tag é composto por um atributo que modifica a tela de impressão caso o usuário (a) queira imprimir a página, a tag <style media="print"> </style> modificará à cor de fundo atual por outra selecionada no momento da impressão.

Nota: Quando um navegador ler uma folha de estilo, ele irá formatar o documento HTML de acordo com as informações da folha de estilo. Se algumas propriedades foram definidas para o mesmo seletor (elemento) em diferentes folhas de estilo, será usado o valor da última folha de estilo lida.
Dica: Para vincular a uma folha de estilo externa, use a tag <link>.

Maneiras de inserir a tag <style> </style>:

                    <!DOCTYPE html>
                    <html lang="pt-br">
                        <head>
                            <title>...</title>
                                <style>
                                    body {
                                        backgrownd-color: red;
                                    }
                                </style>

                                        IMPORTANTE: abre-se uma nova tag <style> </style> para inserção de configuração da cor de impressão da página.

                                <style media="print">
                                    body {
                                        backgrownd-color: yellow;
                                    }
                                </style>
                        </head>

                        <body>

                        </body>
                    </html>

Atributos Globais

A tag <style> </style> suporta todos os Atributos Globais HTML.

accesskey, class, contenteditable, data-*, dir, draggable, hidden, id, lang, spellcheck, style, tabindex, title, translate.
