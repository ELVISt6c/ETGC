<STYLE>

Definição

A tag <style> </style> define o estilo (CSS) em um documento (HTML).

Dentro do elemento deve ser especificado os termos (HTML) que serão renderizados em um navegador.

A tag abrange folhas de estilo para uma página web.

Importante: Há dois tipos de tag <style> </style> em um documento HTML.

Importante: O segundo tipo de tag é composto por um atributo que modifica a tela de impressão caso o usuário (a) queira imprimir a página, <style media="print"> </style> modificando assim a cor de fundo.

Maneiras de inserir a tag <style> </style>:

                    <!DOCTYPE html>
                    <html lang="pt-br">
                        <head>
                            <title>...</title>
                                <style>
                                    body {
                                        backgrownd-color: red;
                                    }
                                </style>

                                        IMPORTANTE: abre-se uma nova tag <style> para inserção de configuração da cor de impressão da página.

                                <style media="print">
                                     body {
                                        backgrownd-color: yellow;
                                    }
                                </style>
                        </head>

                        <body>

                        </body>
                    </html>

Atributos Globais

A tag <body> </body> suporta todos os Atributos Globais HTML.

accesskey, class, contenteditable, data-*, dir, draggable, hidden, id, lang, spellcheck, style, tabindex, title, translate.
