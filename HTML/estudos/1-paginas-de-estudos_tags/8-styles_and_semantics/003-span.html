<SPAN>

Definição

O elemento <span> é um contêiner inline genérico para conteúdo de frase, que não representa nada importante. Ele pode ser usado para agrupar elementos para fins de estilo

A tag <span> </span> define um contêiner em um documento sendo usado para conteúdo de texto ou partes de um documento (HTML).

A tag <span> </span> pode ser estilizada facilmente com (CSS) ou manipulada com (JS) através dos atributos class="" ou id="".

A tag <span> </span> é parecida com a tag <div> </div> sendo que esta é um elemento embutido enquanto <div> </div> é um elemento de nível bloco.

Maneiras de inserir a tag <span> </span>:

                    <!DOCTYPE html>
                    <html lang="pt-br">
                        <head>
                            <title>...</title>
                            <style>
                                .cor {
                                    color: blue;
                                }
                            <style>
                        </head>
                        <body>
                            <p>Uma característica da espan é poder usa-lá para mudar a cor de uma palavra sem 
                            comportamento semântico, por exemplo: <span class="cor">AZUL</span><p>
                        </body>
                    </html>

Atributos Globais

A tag <span> </span> suporta todos os Atributos Globais HTML.

accesskey, class, contenteditable, data-*, dir, draggable, hidden, id, lang, spellcheck, style, tabindex, title, translate.
