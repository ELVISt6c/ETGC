<P>

Definição

A tag <p> cria um parágrafo para preenchimento de texto.

Os navegadores reconhecem a tag "p" não só como um parágrado, mas com uma quebra de linha também.

Uma linha em branco é adicionada antes e depois do elemento, fazendo com que este tenha a quebra automática sem a necessidade de uso da tag <br> para quebrar uma linha.

Dica: Use CSS para dar mais estilização aos parágrafos, que por muiitas vezes ficam de lado.

Maneiras de inserir a tag "p" Paragraph:

                            <body>
                                <h1>Paragraph</h1>
                                    <p>Uma linha escrita para demonstração do conteúdo.<p>
                            </body>

Atributos Globais

A tag <p> suporta todos os atributos globais.

Atributos: accesskey, autocapitalize, class, contenteditable, contextmenu, data-*, dir, draggable, hidden, id, itemprop, lang, slot, style, tabindex, title, translate.
