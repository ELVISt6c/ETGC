<BR>

Definição

A tag <br> cria uma quebra entre linhas.

Nunca utilize essa tag para realizar quebra entre parágrafos.

Pode ser utilizada para escrever poemas e endereços.

Atenção: Essa tag insere uma única quebra de linha.

Maneiras de inserir a tag "br":

                            <body>
                                <p>Uma amostra da tag <br> para melhor entender!</p>
                                    <br>
                            </body>

Uma amostra da tag <br> para melhor entender!


Esse trecho mostra à quebra.

Atributos Globais

A tag <br> suporta todos os Atributos Globais HTML.

accesskey, class, contenteditable, data-*, dir, draggable, hidden, id, lang, spellcheck, style, tabindex, title, translate.
