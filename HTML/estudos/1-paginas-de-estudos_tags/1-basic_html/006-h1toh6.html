<H1 TO H6>

Definição

Aa tags <h1> até <h6> são usadas para criar títulos em uma página HTML.

Uma ordem hierárquica é seguida a começar de "h1" que define o título mais importante até "h6" que define o título menos importante.

Observação: use apenas um <h1> por página - deve representar o título / assunto principal de toda a página. Além disso, não pule os níveis de título - comece com <h1>, depois use <h2>e assim por diante.

Maneiras de inserir as tags de h1 até h6:

                            <body>
                                <h1>Text</h1> 
                                <h2>Text</h2>  
                                <h3>Text</h3> 
                                <h4>Text</h4>  
                                <h5>Text</h5> 
                                <h6>Text</h6>  
                            </body>

Atributos Globais

A tag <body> suporta todos os atributos globais.

Atributos: accesskey, autocapitalize, class, contenteditable, contextmenu, data-*, dir, draggable, hidden, id, itemprop, lang, slot, style, tabindex, title, translate.
