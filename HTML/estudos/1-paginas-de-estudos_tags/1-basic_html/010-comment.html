<!--...-->

Definição

A tag <!--...--> é utilizada para aplicar comentários no código fonte.

Os comentários não são exibidos no navegador.

Você pode usar comentários para explicar seu código, o que pode ajudá-lo quando você editar o código-fonte em uma data posterior. Isso é especialmente útil se você tiver muitos códigos.

Dica: Você pode usar a tag de comentário para "ocultar" scripts de navegadores sem suporte para scripts (para que eles não os mostrem como texto simples).

Importante: As duas barras no final da linha de comentário (//) é o símbolo de comentário JavaScript. Isso evita que o JavaScript execute a tag ->.

Maneiras de inserir a tag "<!---->" comment:

                            <body>
                                <h1>Um exemplo comment</h1>
                                    <p>Esse exemplo mostra algo à ser comentado.</p>
                                        <!--Aqui se aplica o comentário que será feito--> 
                                <p>Sem mudança! O comentário não altera nada.</p>
                            </body>
                            

                            <body>
                                <h1>Um exemplo comment em JavaScript</h1>
                                    <script type="text/javascript">
                                        <!--
                                        function displayMsg() {
                                        alert("Hello World!")
                                        }
                                        //-->
                                    </script>
                            </body>

Atributos Globais

A tag de comentário não oferece suporte a nenhum atributo padrão.
