<HTML>

Definição

A tag <html>...</html> representa a raiz de todo documento.

A tag <html>...</html> é um "container" para tudo que será inserido na página.

Também pode ser adicionado o atributo lang="" que determina em qual liguagem nativa à página estará sendo editada.

Maneiras de inserir a tag <html>:

                    <html lang="pt-br">
                        <head>
                            <title>...</title>
                        </head>

                        <body>
                        </body>
                    </html>

Atributos Globais

A tag <html> </html> suporta todos os Atributos Globais HTML.

accesskey, class, contenteditable, data-*, dir, draggable, hidden, id, lang, spellcheck, style, tabindex, title, translate.
