"BUTTON"

Definição

A input <button> Cria um botão de comportamento padrão! é utilizado para enviar e redefinir formulários. Para fazer botões fazerem qualquer coisa, é necesário escrever código JavaScript.

Maneiras de inserir a input "button" Botão:

                            <input type="button" name="button" value="button" id"button"=>

Atributos Globais

A input <button> suporta todos os Atributos Globais HTML.

accesskey, class, contenteditable, data-*, dir, draggable, hidden, id, lang, spellcheck, style, tabindex, title, translate.

Atributos Input

type, name, value.
