<BODY>

Definição

A tag <body> define o corpo principal da página.

O elemento contém todo conteúdo de uma página HTML.

a tag abrange títulos, imagens, links, parágrafos, tabelas, hiperlkinks, listas, etc.

Importante:Só pode haver um elemento "body" em um documento HTML.

Maneiras de inserir a tag body:

                            <!DOCTYPE html>
                            <html lang="pt-br">
                                <head>
                                    <title>...</title>
                                </head>

                                <body>
                                    <h1>...</h1>
                                </body>
                            </html>

Atributos Globais

A tag <body> suporta todos os atributos globais.

Atributos: accesskey, autocapitalize, class, contenteditable, contextmenu, data-*, dir, draggable, hidden, id, itemprop, lang, slot, style, tabindex, title, translate.
