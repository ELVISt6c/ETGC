<HTML>

Definição

A tag <html> representa a raiz de todo documento "html".

A tag html é um "container" para tudo que será inserido na página.

Também pode ser adicionado o atributo "lang" que determina em qual liguagem nativa à página estará sendo editada.

Maneiras de inserir a tag html:

                            <html lang="pt-br">
                                <head>
                                    <title>...</title>
                                </head>

                                <body>
                                </body>
                            </html>

Atributos Globais

A tag <html> suporta todos os atributos globais.

Atributos: accesskey, autocapitalize, class, contenteditable, contextmenu, data-*, dir, draggable, hidden, id, itemprop, lang, slot, style, tabindex, title, translate.
