<HEAD>

Definição

A tag <head> é um container para o recebimento de metadados.

Metadados são (dados sobre dados), que são dados sobre o documento em html, e que não são mostrados para leitores de tela.

Metadados em geral define o título de um documento, conjunto de caractéres, estilos scripts, entre outras matainformações.

Maneiras de inserir a tag head e seus elementos:

                            <head>
                                <title> </title>
                                <style> </style>
                                <base> 
                                <link>
                                <meta> 
                                <script> </script>
                                <noscript> </noscript>
                            </head>

Atributos Globais

A tag <html> suporta todos os atributos globais.

Atributos: accesskey, autocapitalize, class, contenteditable, contextmenu, data-*, dir, draggable, hidden, id, itemprop, lang, slot, style, tabindex, title, translate.
