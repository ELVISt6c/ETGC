CONTAINER
    Classe definada para dar ênfase ao bloco raiz.

BOX
    Classe definida para dar ênfase ao bloco filho de CONTAINER, e contém a página que abrange todo conteúdo.

BAR
    Classe definida para dar ênfase a "barra".

NAV-BAR
    Classe definada para dar ênfase a "barra de navegação" e antecedente de "barra"

ETGC
    Classe relacionada à logo-marca e tipo do site.

ESTUDOS
    Classe que define um botão de navegação.

MATERIAIS
    Classe que define um botão de navegação.

PORTFOLIO
    Classe que define um botão de navegação.

TAG-HTML
    Classe que define o título da página HTML.

PRINCIPAL
    Classe definada para dar ênfase ao contéudo principal da página.

PAGINAÇÃO
    Classe definida para dar ênfase aos links de navegação de páginas aleatórias.

PAGINA-NAVI
    Classe principal para links de navegação.

ROLE
    Atribudo que define aos navegadores e mecânismos de pesquisa (SEO), a finalidade do bloco inserido e com peso semântico.

PAGINAS
    Classe definida para a contagem do número de páginas totais no documento HTML.

PRIMEIRA-PAGINA 
    Classe definida para dar volta única para à primeira página.

ARIA-LABEL
    Atributo que define dentro do elemento a funcionalidade de "atual" em forma de leitura somente e com peso semântico.

PULAR-PAGINA    
    Classe definida para tanto retroceder páginas quanto pular páginas.

PROXIMA-PAGINA  
    Classe definida para passar uma pagina.

PAGINA-ANTERIOR
    Classe definida para voltar uma página.

ULTIMA-PAGINA
    Classe definida para dar entrada única à última página.

HN
    Classe dedinida para dar ênfase ao conjundo de tags "h1" ao "h6".
